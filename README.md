# UE5 prototype: tile-based third-person game

About:

An early-stage project featuring the following:
- procedural level generation
- player movement system
- multiplayer system with built-in support for up to 4 players
- early destruction/combat mechanics

How to run:

Open Game.uproject in Unreal Engine 5.6
Load Blank.umap level
Run as either Standalone, Listen Server or Client
